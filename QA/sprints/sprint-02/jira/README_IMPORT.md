# Jira Import - Sprint 02

## Archivo
- `JIRA_IMPORT_SPRINT2.csv`

## Pasos de importacion
1. Ir a Jira > Filters > Advanced issue search.
2. Seleccionar `Import issues from CSV`.
3. Cargar `JIRA_IMPORT_SPRINT2.csv`.
4. Mapear campos:
   - Issue Type -> Issue Type
   - Summary -> Summary
   - Description -> Description
   - Priority -> Priority
   - Labels -> Labels
5. Confirmar importacion y asignar Sprint 02.
